🎛️🐻 THE CHOPPA by MATTBEAR
CHOP. PLAY. REMIX.

The Choppa is a browser-based auto-chop pad drummer and sample player.

Load audio, slice it into pads, play it with touch, mouse, keyboard, or MIDI, then shape it with FX knobs.

No install. No subscription. No BS. Just browser noise.

✅ Loads audio loops

✅ Plays with touch, mouse, keyboard, or MIDI
✅ Supports Launchkey Mini pad mapping
✅ Includes Smart Chop and Grid Chop

🎧 SUPPORTED AUDIO

The Choppa works with most common browser-supported audio files, including:

• MP3
• WAV
• M4A / AAC
• OGG
• Some FLAC files
• Some WEBM audio files

Best choices:

• WAV = best for clean chopping
• MP3 = easiest compatibility

If a file does not load, try converting it to MP3 or WAV.

✂️ CHOP MODES

Grid Chop 16
Grid Chop cuts the loaded loop into 16 equal pieces.

This is the safest and most predictable chop mode.

Smart Chop 16
Smart Chop listens for sudden loud hits in your loop and automatically places the 16 slices around those hit points.

Best for:

• Drum loops
• Beat loops
• Percussion
• Stabs
• Samples with clear attacks

If Smart Chop sounds strange, use Grid Chop 16 instead.

▶️ BASIC SETUP

Press Start Audio.
Press Load Loop.
Choose an audio file.
Press Grid Chop 16 for even slices.
Press Smart Chop 16 for hit-based slices.
Press the pads to play your chopped loop.
Use the FX knobs to change the sound.
Press Panic Stop if audio keeps playing or gets too loud.

🎼 KEYBOARD SAMPLE BANK

The keyboard/sample bank lets you load one or more audio samples and play them across the on-screen keyboard, computer keys, or MIDI notes.

How to use it

Press Start Audio.
Press Load Key Samples.
Choose one sample to pitch across the keyboard.
Or choose multiple samples to map across notes.
Set Root Note if you want the first sample to start on a specific MIDI note.
Choose Sample Bank to play mapped samples.
Choose Pitch First Sample to stretch one sample across the keyboard.
Choose Synth Only to use the built-in synth sound.
Play using the on-screen keyboard, computer keys, or MIDI controller.

🛠️ OTHER CONTROLS

🔁 Reverse Pads
Flips the slice order.

Pad 1 becomes Pad 16.
Pad 16 becomes Pad 1.

🥁 Load Demo Loop
Loads a built-in demo beat so you can test the app without adding your own file.

💾 Export Pad Map
Downloads a small JSON file showing which pad triggers which slice.

🔊 Volume
Controls overall output level.

🎵 Pitch
Raises or lowers playback pitch.

⚡ Attack
Controls how quickly the sound starts.

🌫️ Release
Controls how quickly the sound fades out.

🧹 Filter
Darkens or brightens the sound.

✋ Choke
When on, each new pad stops the previous pad.

Good for tight drum-machine style playing.

▶️ Play Mode
One-shot plays the full slice.

Gate-ish short plays a shorter version.

🎯 Velocity
MIDI velocity uses how hard you hit the controller.

Fixed full plays every hit at full strength.

🧯 TROUBLESHOOTING

❌ No sound
Press Start Audio first.

Browsers block sound until you interact with the page.

❌ Loop will not load
Try MP3 or WAV.

Some audio files do not decode correctly in browsers.

❌ Smart Chop sounds strange
Use Grid Chop 16 instead.

❌ MIDI does not work
Use Chrome or Edge.

Press Enable MIDI.

Allow MIDI permission.

❌ Too much sound is playing
Press Panic Stop.
