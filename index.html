Instructions

Start quickly

  1. Press Defaults to load the demo and reset controls.
  2. Use keys 1-8 and Q-I or click the pads.
  3. Watch each pad’s bottom-right time label to confirm the slice start.

Use your own audio

  1. Press Upload Audio.
  2. Choose a song or loop.
  3. Use Smart Chop for detected phrases or Grid Chop for exact divisions.

MIDI keyboard safety

  1. Click THE CHOPPA five times.
  2. Set Keyboard Safe Octave.
  3. Piano-key notes in that octave are ignored while pads still work.

Troubleshooting

Sounds start from beginning

  • Check that mode is not Full Loop.
  • Use Grid Chop to test exact slice offsets.
  • Pad labels should show different start times like 0.00s, 0.25s, 0.50s.

No sound

  • Press Start Audio.
  • Check browser volume and system output.
  • Press Defaults to load a known working demo.

MIDI keys trigger things

  • Open admin with five clicks on THE CHOPPA.
  • Set Keyboard Safe Octave.
  • Save admin settings.

FAQ

What does Defaults do?

It loads the demo loop, selects Smart Chop, sets Fixed Velocity, turns Choke On, returns to the saved default mode, and resets every FX knob visually and sonically.

What does Clear do?

Clear stops active sound, stops Full Loop, removes slice loops, clears loaded audio, clears held gate notes, and resets the upload status.

How do I know slicing works?

Use Grid Chop. Each pad shows a different start time. If you hear the same start every time, you are in Full Loop mode or the page is stale.
